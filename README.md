# Meu PortfolioHUB - Desafio Bootcamp CEUB

[cite_start]Este é o projeto da "Entrega Final: Implantação PortfolioHUB + IA GEMINI" [cite: 6] [cite_start]do Bootcamp I do CEUB[cite: 1, 2].

## [cite_start]1. Plano de Implantação 

* [cite_start]**Objetivo:** Criar um site simples de portfólio (PortfolioHUB) [cite: 13] com 3 páginas: Início, Sobre e Projetos.
* **Tecnologia:** HTML e CSS puros.
* [cite_start]**Hospedagem (Produção):** O site será "lançado em produção"  usando o GitHub Pages.
* [cite_start]**IA de Apoio:** O Google Gemini [cite: 15] foi usado para guiar todo o processo.

## [cite_start]2. Integração Git/Github 

* Este repositório armazena todo o código do site.
* [cite_start]A página "Projetos" será integrada ao GitHub  listando links para meus outros repositórios.

## [cite_start]3. Gestão e Segurança 

* [cite_start]**Gestão de Usuários:** A gestão é feita pelo dono do repositório no GitHub.
* [cite_start]**Segurança:** Foi implementada uma política de "Branch Protection Rule"  na branch `main` para exigir Pull Requests.

## [cite_start]4. Compartilhamento e Controle 

* O site é compartilhado publicamente através do link do GitHub Pages.
* [cite_start]O controle de versão é feito pelo Git/GitHub.

## [cite_start]5. Testes 

* Os testes foram feitos acessando o link público do GitHub Pages e verificando se todas as páginas e links funcionam.

## [cite_start]6. Apresentação 

* [cite_start]A apresentação final será um vídeo no YouTube  e este documento PDF.

# Meu PortfolioHUB - Desafio Bootcamp CEUB

Este é o projeto da "Entrega Final: Implantação PortfolioHUB + IA GEMINI" [cite: 6] [cite_start]do Bootcamp I do CEUB[cite: 1, 2].

## 1. Plano de Implantação 

**Objetivo:** Criar um site simples de portfólio (PortfolioHUB) [cite: 13] com 3 páginas: Início, Sobre e Projetos.
**Tecnologia:** HTML e CSS puros.
**Hospedagem (Produção):** O site será "lançado em produção"  usando o GitHub Pages.
**IA de Apoio:** O Google Gemini [cite: 15] foi usado para guiar todo o processo.

## 2. Integração Git/Github 

* Este repositório armazena todo o código do site.
* A página "Projetos" será integrada ao GitHub  listando links para meus outros repositórios.

## 3. Gestão e Segurança 

**Gestão de Usuários:** A gestão é feita pelo dono do repositório no GitHub.
**Segurança:** Foi implementada uma política de "Branch Protection Rule"  na branch `main` para exigir Pull Requests.

## 4. Compartilhamento e Controle 

* O site é compartilhado publicamente através do link do GitHub Pages.
* O controle de versão é feito pelo Git/GitHub.

## 5. Testes 

* Os testes foram feitos acessando o link público do GitHub Pages e verificando se todas as páginas e links funcionam.

## 6. Apresentação 

* A apresentação final será um vídeo no YouTube  e este documento PDF.
